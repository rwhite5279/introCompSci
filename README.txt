introCompSci README.txt

The introCompSci repository contains files related to the Introduction to Computer Science course at Polytechnic School, Pasadena, CA.

The course is a Python-based introduction to some of the basic data structures, control structures, algorithms, and analysis techniques that are the foundation of any understanding of computer science.

This course is offered as a Spring Semester (13-week) elective course at Polytechnic, and was developed by Richard White. Further information about the course is available in some of the files included in this repository.

Not all files are available in this repository; some copyrighted materials that are used in the classroom under the Fair Use doctrine of copyright law are obviously not included here.

Some of the files/directories include:

.
|____admin/ - administrative files, including Terms of Service, course description, etc.
|____assignments/ - PDFs of major assignments, text files of many homeworks
|____faqs.html    - with contact information, etc.
|____images/      - various images used in the main site
|____index.html   - the home page
|____lectures/    - html-based lectures and graphics
|____links.html   - both local and external links to related materials, content, etc.
|____misc/        - stuff that doesn't go anywhere else
|____README.txt   - this file
|____styles_compsci.css - CSS rules for the site
